## Properties

### **Presets** ([Presets](../enums/Presets))
This property holds all the information about blasters. You can use it to either modify or view blaster data.
Keep in mind that when you change data in the presets, those changes will only apply to new blasters. If you need to update existing blasters, you'll have to manually adjust the [BlasterClient.BlasterStats](./BlasterClient#BlasterStats).

```lua
local BlasterClientConstructor = require("BlasterClient")
BlasterClientConstructor.Presets["E-11"].Firerate = 0.2 -- New E-11 Blasters will have a 0.2 Firerate
print(BlasterClientConstructor.Presets["E-11"].BlasterType) --> Automatic
```

## Methods

### **new** ([BlasterClient](./BlasterClient))
Creates a new [BlasterClient](./BlasterClient) class which allows you to interact with the blaster's functions.

```lua
local BlasterClientConstructor = require("BlasterClient")
game:GetService("Players").PlayerAdded:Connect(function(player)
    player.CharacterAdded:Connect(function(character)
        local BlasterClient = BlasterClientConstructor.new(character, player.Backpack["E-11"])
        print(BlasterClient)
    end)
end)
```

### **Parameters**

- `character` : [Model](https://create.roblox.com/docs/reference/engine/classes/Model)  
- `tool` : [Tool](https://create.roblox.com/docs/reference/engine/classes/Tool)  
- `blasterStats` : [BlasterStats?](../enums/BlasterStats)  

### **Returns**
- **BlasterClient** : [BlasterClient](./BlasterClient)

## **GetBlasterClientFromTool**
Finds an existing [BlasterClient](./BlasterClient) from a [Tool](https://create.roblox.com/docs/reference/engine/classes/Tool).  

```lua
local BlasterClientConstructor = require("BlasterClient")
local BlasterClient = BlasterClientConstructor:GetBlasterClientFromTool(game:GetService("Players").Wyv_n.Backpack["E-11"])
```

### **Parameters**
- `tool` : [Tool](https://create.roblox.com/docs/reference/engine/classes/Tool)  

### **Returns**
